# v0.0.5 (Proactive Delta Context 初版)

Context Delta の npm 版初リリースです。TypeScript CLI、Proactive Delta Context フロー、PromptCard 連携を含むセットアップを提供します。

## 主要機能

- **Node.js CLI** – `bin/delta.ts` とサービス層（assistants/filesystem/templates/context/promptcards）を追加し、`delta init` / `delta update` / `delta card sync` を TypeScript で実装。ロケール判定、各アシスタントへのプロンプト同期、openspec マイグレーション、`--update-root-agents` をサポート。
- **Proactive Delta Context** – `delta-concept` → `delta-roadmap` → `delta-propose` → `delta-verify` → `delta-apply` → `delta-archive` のフローをテンプレ化し、差分のみを一個流しで処理。`delta-verify` は PromptCard の Rubric/Regression を採点し、Pass/Borderline/Fail を判定します。
- **PromptCard エコシステム** – `context-delta/promptcards/` に PRD/SRS/Behavior/Model/API/Data/Architecture/Delivery/Module の 9 カードを同梱。`delta card sync` で `docs/promptcards/registry.(md|json)` を更新し、delta プロンプトがカードを参照できます。
- **ドキュメント拡充** – README・AGENTS.md・`docs/proactive-delta-context.md` にリーン思想と PromptCard カスタマイズ手順を記載。`docs/promptcard.md` でカスタムカードの作り方を解説しています。

## インストール

```
npm install -g context-delta
delta --help
```

Node.js 20 以上が必須です。

## 既知の課題

- `delta propose/apply/archive` は PromptCard 参照手順をテンプレ化済みですが、将来バージョンで CLI レベルの自動適用を追加予定です。
